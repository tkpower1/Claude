## Target Audiences

### Audience 1: Beach & Outdoor Enthusiasts
- Interests: Beach, surfing, hiking, camping, fishing, boating, paddleboarding
- Behaviors: Frequent travelers, outdoor activity apps
- Age: 25-55

### Audience 2: Vacation Planners
- Interests: Travel, vacation rentals, Airbnb, beach destinations, cruise lines
- Behaviors: Engaged shoppers, recently searched travel
- Age: 25-55

### Audience 3: Suncare Buyers
- Interests: Sunscreen brands (Supergoop, Sun Bum, Coppertone), skincare
- Behaviors: Beauty & skincare purchasers
- Age: 22-50

### Audience 4: Festival & Event Goers
- Interests: Music festivals, Coachella, outdoor concerts, sporting events
- Behaviors: Event ticket purchasers
- Age: 21-40

### Audience 5: Parents (Kids Get Burned)
- Interests: Parenting, family travel, kids activities
- Demographics: Parents with children 3-17
- Age: 28-50

### Audience 6: Lookalike
- 1% Lookalike of website visitors (after you get traffic)
- 1% Lookalike of purchasers (after you get sales)

## Meta Pixel Setup

Add this code to your site to track conversions:

1. Go to Meta Business Suite → Events Manager
2. Create a new Pixel
3. Add the base code to your `<head>` (I can add this for you)
4. Set up these events:
   - PageView (automatic)
   - ViewContent (landing page)
   - AddToCart (Stripe click)
   - Purchase (Stripe thank you page)

---

## Budget Recommendation

**Week 1-2 (Testing):** $30-50/day
- Test 3 audiences, 3 ad variations each
- Let each ad get 1,000+ impressions before judging
- Kill ads with CTR < 1% after 2,000 impressions

**Week 3+ (Scaling):** Increase budget on winners
- Double budget on ad sets with CPA < $20
- Add retargeting once you have 500+ site visitors

---

## Success Metrics to Track

| Metric | Target |
|--------|--------|
| CTR (Click-through rate) | > 1.5% |
| CPC (Cost per click) | < $1.50 |
| Landing page conversion | > 3% |
| CPA (Cost per acquisition) | < $25 |
| ROAS (Return on ad spend) | > 1.5x |
